# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
The objective of this task is to explore and visualize the Iris dataset
to understand feature distributions, relationships, and data quality.

## Dataset Used
- Iris Dataset
- Loaded using the seaborn library

## Tools & Libraries
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## Work Done
- Loaded and inspected the dataset
- Checked shape, columns, and summary statistics
- Visualized data using scatter plots, histograms, and box plots
- Analyzed distributions and outliers

## Key Findings
- The dataset contains no missing values
- Clear separation between species is visible
- Features are well distributed and suitable for ML models
